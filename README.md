# Panel Zarządzania Social Media

Prosty panel w Pythonie (Tkinter) do szybkiego otwierania stron logowania oraz podglądu plików z danymi dostępowymi w Notatniku.

## Struktura

```
SocialMediaPanel/
├── main.py
├── data/
│   ├── facebook.txt
│   ├── instagram.txt
│   ├── tiktok.txt
│   └── x.txt
└── .gitignore
```

## Uruchomienie (Windows)

1. Upewnij się, że masz zainstalowany Python.
2. Uruchom aplikację:

```
python main.py
```

## Uwagi

- Przycisk z nazwą platformy otwiera stronę logowania w domyślnej przeglądarce.
- Przycisk "Otwórz dane" otwiera plik tekstowy z danymi logowania w Notatniku.
